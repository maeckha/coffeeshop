This application was designed and implemented as part of a bachelor thesis at the university of applied science Constance. It is intended to support the software quality assurance lecture. The exemplary web shop should offer a functioning and comprehensive software with which the students can work on different types and levels of tests.

The entire source code can be viewed and downloaded from the university's Gitlab server. Only open source libraries and tools were used, so that the application can be run by everyone without restrictions.
